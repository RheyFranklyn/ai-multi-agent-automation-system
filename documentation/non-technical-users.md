# User Documentation (Non-Technical)

## What This System Does

This system is an AI-powered automation tool that helps handle customer messages, generate research reports, and organize daily tasks automatically.

You simply send a message, and the system will handle everything for you.

---

## How It Works (Simple Explanation)

1. You send a message (example: complaint, question, or task)
2. The AI reads and understands your message
3. It automatically decides what type of request it is:
   - Customer Support
   - Research Request
   - Productivity Task
4. The system processes it and sends you a result via email

---

## What You Will Receive

### Support Requests
- A professional reply to your issue
- Sent directly to your email
- Logged for tracking

### Research Requests
- A structured AI-generated report
- Sent as a PDF file

### Productivity Requests
- A clean task list or plan
- Sent as a PDF file

---

## Benefits

- Saves time
- Automates repetitive work
- Reduces manual effort
- Provides instant responses
- Organizes information clearly

---

## What You Need to Do

Nothing complicated.

Just send your message, and the system will handle the rest automatically.
